# P15.32 expdesc Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Port PUC Lua's `expdesc` expression descriptor to Zig codegen, eliminating redundant MOVE/LOADK instructions by deferring value materialization until a register is actually needed.

**Architecture:** Replace `genExp() -> u8` (always materializes to a register) with `genExp() -> ExpDesc` (describes the value's kind: local, constant, relocatable, etc.). Add `dischargevars`/`discharge2reg`/`exp2nextreg` operations that materialize only when needed. Use Zig tagged union for ExpDesc (not C-style enum+union).

**Tech Stack:** Zig 0.16.0, existing `codegen_bc.zig` (~4370 lines), PUC Lua 5.5.0 `lcode.c` as architectural reference.

**Spec:** `docs/superpowers/specs/2026-07-18-p15-32-expdesc-design.md`

---

## File Structure

- **Modify:** `src/lua/codegen_bc.zig` — Add `ExpDesc` struct, discharge functions, convert `genExp` to return `ExpDesc`, adapt all call sites.
- **No new files** — everything fits in the existing codegen file.

## Key Design Decisions

1. **ExpDesc as Zig tagged union** — `ExpDesc` is a `struct` with `kind: Kind` + `u: U` union, matching PUC's `expdesc` but using Zig-native constructs.
2. **Incremental migration** — `genExp` returns `ExpDesc`, but a helper `genExpReg(e) -> u8` calls `genExp` + `exp2nextreg` so existing call sites work unchanged during migration.
3. **Jump lists** — Use `?i32` (PC of first jump) with patch-through (each jump stores next in its B field), matching PUC's `luaK_patchlist`.
4. **No VJMP initially** — Stages 1-3 handle non-jump expressions. `genAndExp`/`genOrExp`/comparisons keep current TEST+JMP approach (already works). VJMP is a later optimization.

```zig
    // -----------------------------------------------------------------------
    // Expression descriptor (PUC Lua expdesc — P15.32)
    // -----------------------------------------------------------------------
    //
    // PUC Lua delays value materialization until a register is actually
    // needed. An ExpDesc describes *what* the value is, not *where* it is.
    // `dischargevars` resolves variable-kinds to value-kinds; `discharge2reg`
    // materializes a value into a specific register; `exp2nextreg` materializes
    // into the next free register.

    const ExpDesc = struct {
        kind: Kind,
        u: U = .{ .info = 0 },
        // Jump lists for short-circuit / conditional expressions.
        // PUC stores these as patch-list head pointers; NULL = empty.
        // We use 0 = empty, positive = pc+1 (so 0 can be a valid pc).
        t_list: i32 = 0,
        f_list: i32 = 0,

        const Kind = enum {
            void,       // empty explist slot
            nil,        // constant nil
            true,       // constant true
            false,      // constant false
            k,          // constant pool index (u.info)
            k_int,      // integer constant (u.ival)
            k_float,    // float constant (u.nval)
            k_str,      // string constant (u.strval)
            non_reloc,  // value in fixed register (u.info = reg)
            local,      // local variable (u.var_.ridx = reg)
            upval,      // upvalue (u.info = index)
            const_local, // compile-time const local (u.info = actvar index)
            indexed,    // t[k] with register key (u.ind)
            index_i,    // t[const_int] (u.ind)
            index_str,  // t["string"] (u.ind)
            index_up,   // upvalue[k] (u.ind)
            reloc,      // relocatable instruction (u.info = pc)
            call,       // function call (u.info = pc)
            vararg,     // vararg expression (u.info = pc)
        };

        const U = union {
            info: i32,
            ival: i64,
            nval: f64,
            strval: []const u8,
            var_: struct { ridx: u8, vidx: i16 = 0 },
            ind: struct { idx: i16, t: u8, ro: bool = false, keystr: i32 = -1 },
        };
    };
```

```zig
    /// Free the register held by an expression if it's a non-relocatable
    /// temporary (above nvarstack). Mirrors PUC `freeexp`.
    fn freeExp(self: *Codegen, e: *const ExpDesc) void {
        if (e.kind == .non_reloc) {
            self.freeReg(@intCast(e.u.info));
        }
    }

    /// Free two expressions in correct high-to-low order. Mirrors PUC `freeexps`.
    fn freeExps(self: *Codegen, e1: *const ExpDesc, e2: *const ExpDesc) void {
        const r1: i32 = if (e1.kind == .non_reloc) e1.u.info else -1;
        const r2: i32 = if (e2.kind == .non_reloc) e2.u.info else -1;
        if (r1 > r2) {
            self.freeExp(e1);
            self.freeExp(e2);
        } else {
            self.freeExp(e2);
            self.freeExp(e1);
        }
    }
```

- [ ] **Step 3: Add dischargevars**

Add after freeExps. This is the core PUC `luaK_dischargevars` — resolves variable-kinds to value-kinds without materializing to a register:

```zig
    /// Resolve variable-kinds (VLOCAL, VUPVAL, VINDEXED*, VCONST) to
    /// value-kinds (VNONRELOC, VRELOC, VK*, VNIL, VTRUE, VFALSE).
    /// Does NOT allocate a register — the result is either a constant
    /// or a reference to an existing instruction/register.
    /// Mirrors PUC Lua `luaK_dischargevars`.
    fn dischargeVars(self: *Codegen, e: *ExpDesc) Error!void {
        switch (e.kind) {
            .local => {
                // Local becomes non-relocatable: value is in a fixed register.
                const reg = e.u.var_.ridx;
                e.u.info = reg;
                e.kind = .non_reloc;
            },
            .upval => {
                // Emit GETUPVAL with A=0 (relocatable), patch later by discharge2reg.
                const pc = try self.builder.emitABC(.getupval, 0, @intCast(e.u.info), 0, self.line_hint);
                e.u.info = @intCast(pc);
                e.kind = .reloc;
            },
            .index_up => {
                const pc = try self.builder.emitABC(.gettabup, 0, e.u.ind.t, @intCast(e.u.ind.idx), self.line_hint);
                e.u.info = @intCast(pc);
                e.kind = .reloc;
            },
            .index_i => {
                self.freeReg(e.u.ind.t);
                const pc = try self.builder.emitABC(.geti, 0, e.u.ind.t, @intCast(e.u.ind.idx), self.line_hint);
                e.u.info = @intCast(pc);
                e.kind = .reloc;
            },
            .index_str => {
                self.freeReg(e.u.ind.t);
                const pc = try self.builder.emitABC(.getfield, 0, e.u.ind.t, @intCast(e.u.ind.idx), self.line_hint);
                e.u.info = @intCast(pc);
                e.kind = .reloc;
            },
            .indexed => {
                self.freeReg2(@intCast(e.u.ind.t), @intCast(e.u.ind.idx));
                const pc = try self.builder.emitABC(.gettable, 0, e.u.ind.t, @intCast(e.u.ind.idx), self.line_hint);
                e.u.info = @intCast(pc);
                e.kind = .reloc;
            },
            .call => {
                // Already returns 1 result; becomes non-relocatable.
                const pc: usize = @intCast(e.u.info);
                e.u.info = @intCast(self.builder.code.items[pc].a);
                e.kind = .non_reloc;
            },
            .vararg => {
                // Set C=2 (one result), becomes relocatable.
                const pc: usize = @intCast(e.u.info);
                self.builder.code.items[pc].c = 2;
                e.kind = .reloc;
            },
            .const_local => {
                // TODO: resolve compile-time const to k_int/k_float/k_str/k.
                // For now, treat as non-reloc (the local's register).
                e.kind = .non_reloc;
            },
            // nil, true, false, k, k_int, k_float, k_str, non_reloc, reloc, void
            // are already value-kinds — nothing to do.
            else => {},
        }
    }
```

- [ ] **Step 4: Add discharge2reg**

Add after dischargeVars. Materializes a value into a specific register:

```zig
    /// Materialize expression `e` into register `reg`.
    /// After this call, `e.kind == .non_reloc` and `e.u.info == reg`.
    /// Mirrors PUC Lua `discharge2reg`.
    fn discharge2reg(self: *Codegen, e: *ExpDesc, reg: u8) Error!void {
        try self.dischargeVars(e);
        switch (e.kind) {
            .nil => {
                _ = try self.builder.emitABC(.loadnil, reg, 0, 0, self.line_hint);
            },
            .false => {
                _ = try self.builder.emitABC(.loadfalse, reg, 0, 0, self.line_hint);
            },
            .true => {
                _ = try self.builder.emitABC(.loadtrue, reg, 0, 0, self.line_hint);
            },
            .k_str => {
                // Convert to pool constant, then LOADK.
                const kid = try self.builder.internString(e.u.strval);
                e.u.info = @intCast(kid);
                e.kind = .k;
                try self.emitLoadK(reg, @intCast(e.u.info), self.line_hint);
            },
            .k => {
                try self.emitLoadK(reg, @intCast(e.u.info), self.line_hint);
            },
            .k_int => {
                if (e.u.ival >= -32768 and e.u.ival <= 32767) {
                    const bits: u32 = @bitCast(@as(i32, @intCast(e.u.ival)));
                    const lo: u8 = @truncate(bits);
                    const hi: u8 = @truncate(bits >> 8);
                    _ = try self.builder.emitABC(.loadi, reg, lo, hi, self.line_hint);
                } else {
                    const kid = try self.builder.internConst(.{ .int = e.u.ival });
                    try self.emitLoadK(reg, kid, self.line_hint);
                }
            },
            .k_float => {
                const kid = try self.builder.internConst(.{ .num = e.u.nval });
                try self.emitLoadK(reg, kid, self.line_hint);
            },
            .reloc => {
                // Patch the instruction's A field to target `reg`.
                const pc: usize = @intCast(e.u.info);
                self.builder.code.items[pc].a = reg;
            },
            .non_reloc => {
                if (reg != e.u.info) {
                    _ = try self.builder.emitABC(.move, reg, @intCast(e.u.info), 0, self.line_hint);
                }
            },
            else => {
                // void, local, upval, const_local, indexed*, call, vararg
                // should have been resolved by dischargeVars. If we get here,
                // it's a bug.
                self.setDiag(.{ .start = 0, .end = 0, .line = self.line_hint, .col = 0 }, "internal: cannot discharge expression to register");
                return error.CodegenError;
            },
        }
        e.u.info = reg;
        e.kind = .non_reloc;
    }
```

```zig
    /// Materialize expression into the next free register (freereg).
    /// Allocates a register and discharges the expression into it.
    /// Mirrors PUC Lua `luaK_exp2nextreg`.
    fn exp2nextreg(self: *Codegen, e: *ExpDesc) Error!u8 {
        try self.dischargeVars(e);
        // Free the register if it's a non-relocatable temp, so we can
        // reuse it if it's at the top of the stack.
        self.freeExp(e);
        const reg = try self.allocReg();
        try self.discharge2reg(e, reg);
        return reg;
    }

    /// Materialize expression into any register. If already in a
    /// non-relocatable register, reuse it. Otherwise allocate a new one.
    /// Mirrors PUC Lua `luaK_exp2anyreg`.
    fn exp2anyreg(self: *Codegen, e: *ExpDesc) Error!u8 {
        try self.dischargeVars(e);
        switch (e.kind) {
            .non_reloc => return @intCast(e.u.info),
            .k_int, .k_float, .k_str, .k, .nil, .true, .false => {
                // Constants: allocate a register.
                const reg = try self.allocReg();
                try self.discharge2reg(e, reg);
                return reg;
            },
            else => {
                // reloc, call, vararg: discharge to next free register.
                return try self.exp2nextreg(e);
            },
        }
    }

    /// Discharge to a value (no register required). Used when we only
    /// need the value for a test/condition, not as a register operand.
    /// Mirrors PUC Lua `luaK_exp2val`.
    fn exp2val(self: *Codegen, e: *ExpDesc) Error!void {
        try self.dischargeVars(e);
    }
```

```zig
    /// Generate code for an expression, returning an ExpDesc instead of
    /// a register. This is the PUC Lua `expr` equivalent. It defers
    /// materialization: the caller decides when to discharge to a register
    /// via exp2nextreg/exp2anyreg/discharge2reg.
    ///
    /// Currently handles leaf expressions (constants, names). Binary ops,
    /// calls, and other complex expressions still use the old genExp path
    /// and return a non_reloc ExpDesc.
    fn genExpDesc(self: *Codegen, e: *const ast.Exp) Error!ExpDesc {
        switch (e.node) {
            .Nil => return .{ .kind = .nil },
            .True => return .{ .kind = .true },
            .False => return .{ .kind = .false },
            .Integer => {
                const lexeme = e.span.slice(self.source);
                const parsed: i64 = parseIntegerLiteral(lexeme) orelse {
                    self.setDiag(e.span, "invalid integer literal");
                    return error.CodegenError;
                };
                return .{ .kind = .k_int, .u = .{ .ival = parsed } };
            },
            .Number => {
                const lexeme = e.span.slice(self.source);
                const val = std.fmt.parseFloat(f64, lexeme) catch {
                    self.setDiag(e.span, "invalid number literal");
                    return error.CodegenError;
                };
                return .{ .kind = .k_float, .u = .{ .nval = val } };
            },
            .String => {
                const lexeme = e.span.slice(self.source);
                const decoded = try self.decodeStringLexeme(lexeme);
                return .{ .kind = .k_str, .u = .{ .strval = decoded } };
            },
            .Name => |n| {
                return self.genNameExpDesc(n.span, n.slice(self.source));
            },
            .Paren => |inner| {
                return self.genExpDesc(inner);
            },
            else => {
                // Fallback: use old genExp, wrap result as non_reloc.
                const reg = try self.genExp(e);
                return .{ .kind = .non_reloc, .u = .{ .info = reg } };
            },
        }
    }
```

```zig
    /// Resolve a name to an ExpDesc (PUC Lua `singlevar`).
    /// Local → VLOCAL, upvalue → VUPVAL, global → VINDEXUP/VINDEXSTR.
    /// Does NOT emit MOVE — the caller discharges only if needed.
    fn genNameExpDesc(self: *Codegen, span: ast.Span, name: []const u8) Error!ExpDesc {
        // Local variable?
        if (self.lookupLocalBinding(name)) |binding| {
            // Check if a global declaration shadows this local.
            if (self.latestDeclaredGlobalDepthSelf(name)) |global_depth| {
                if (global_depth > binding.depth) {
                    // Global: _ENV[name]
                    const name_kid = try self.builder.internString(name);
                    const env_idx = try self.ensureEnvUpvalue();
                    return .{
                        .kind = .index_up,
                        .u = .{ .ind = .{ .idx = @intCast(name_kid), .t = env_idx, .keystr = @intCast(name_kid) } },
                    };
                }
            }
            return .{ .kind = .local, .u = .{ .var_ = .{ .ridx = binding.reg } } };
        }
        // Forced global?
        if (self.isForcedGlobalName(name)) {
            try self.checkDeclaredGlobal(span, name);
            const name_kid = try self.builder.internString(name);
            const env_idx = try self.ensureEnvUpvalue();
            return .{
                .kind = .index_up,
                .u = .{ .ind = .{ .idx = @intCast(name_kid), .t = env_idx, .keystr = @intCast(name_kid) } },
            };
        }
        // Upvalue?
        if (self.upvalues.get(name)) |idx| {
            return .{ .kind = .upval, .u = .{ .info = @intCast(idx) } };
        }
        // Try to capture from outer scope.
        if (self.outer != null) {
            if (self.ensureUpvalue(name)) |idx| {
                return .{ .kind = .upval, .u = .{ .info = @intCast(idx) } };
            } else |_| {}
        }
        // Global: _ENV[name]
        try self.checkDeclaredGlobal(span, name);
        const name_kid = try self.builder.internString(name);
        const env_idx = try self.ensureEnvUpvalue();
        return .{
            .kind = .index_up,
            .u = .{ .ind = .{ .idx = @intCast(name_kid), .t = env_idx, .keystr = @intCast(name_kid) } },
        };
    }
```

Replace the operand compilation in `genBinOp` (lines 1355-1431). The key change: use `genExpDesc` for LHS and RHS, discharge to registers only when needed, and write the result directly to the destination register.

```zig
    fn genBinOp(self: *Codegen, n: anytype, line: u32) Error!u8 {
        // PUC discharges the left expression when it sees the infix operator.
        const lhs_start_pc: usize = @intCast(self.builder.pc());
        var lhs_ed = try self.genExpDesc(n.lhs);
        const lhs_end_pc: usize = @intCast(self.builder.pc());
        for (self.builder.lineinfo.items[lhs_start_pc..lhs_end_pc]) |*inst_line| {
            inst_line.* = line;
        }

        // --- K/I-variant optimization ---
        // Check if RHS is a numeric constant before materializing it.
        const rhs_const = self.numericConstFromExp(n.rhs);

        // Arithmetic / bitwise: try K/I-variant first, then register/register.
        if (binOpToBc(n.op)) |op| {
            // Discharge LHS to a register (needed for all binops).
            const lhs_reg = try self.exp2anyreg(&lhs_ed);

            if (rhs_const) |nc| {
                // Try K/I-variant: R[dst] = R[lhs] <op> K/I
                if (try self.tryEmitConstBinOp(n.op, lhs_reg, nc, line)) |dst| {
                    return dst;
                }
            }

            // Standard register/register path.
            var rhs_ed = try self.genExpDesc(n.rhs);
            const rhs_reg = try self.exp2anyreg(&rhs_ed);
            self.freeExps(&lhs_ed, &rhs_ed);
            const dst = try self.allocReg();
            _ = try self.builder.emitABC(op, dst, lhs_reg, rhs_reg, line);
            return dst;
        }

        // Comparison: produce a boolean value.
        if (n.op == .EqEq or n.op == .NotEq or n.op == .Lt or
            n.op == .Lte or n.op == .Gt or n.op == .Gte)
        {
            var rhs_ed = try self.genExpDesc(n.rhs);
            const lhs_reg = try self.exp2anyreg(&lhs_ed);
            const rhs_reg = try self.exp2anyreg(&rhs_ed);
            self.freeExps(&lhs_ed, &rhs_ed);
            return self.genComparison(n.op, lhs_reg, rhs_reg, line);
        }

        // Concat
        if (n.op == .Concat) {
            var rhs_ed = try self.genExpDesc(n.rhs);
            const lhs_reg = try self.exp2nextreg(&lhs_ed);
            const rhs_reg = try self.exp2nextreg(&rhs_ed);
            _ = try self.builder.emitABC(.concat, lhs_reg, 2, 0, line);
            self.freeReg(rhs_reg);
            return lhs_reg;
        }

        self.setDiag(.{ .start = 0, .end = 0, .line = line, .col = 0 }, "unsupported binary operator");
        return error.CodegenError;
    }
```

```zig
    /// Store an ExpDesc value into an assignment target (PUC `luaK_storevar`).
    /// For local targets: discharge2reg directly into the local's register.
    /// For global/field/index targets: discharge to a temp, then SET.
    fn genSetDesc(self: *Codegen, lhs: *const ast.Exp, val: *ExpDesc, line: u32) Error!void {
        switch (lhs.node) {
            .Name => |n| {
                const name = n.slice(self.source);
                if (self.isForcedGlobalName(name)) {
                    try self.checkDeclaredGlobal(lhs.span, name);
                    if (self.isConstGlobal(name)) {
                        self.setDiagFmt(lhs.span, "attempt to assign to const variable '{s}'", .{name});
                        return error.CodegenError;
                    }
                    const reg = try self.exp2anyreg(val);
                    const kid = try self.builder.internString(name);
                    try self.emitGlobalSet(kid, reg, line);
                    return;
                }
                if (self.lookupLocal(name)) |reg| {
                    if (self.isReadonlyLocal(reg)) {
                        self.setDiagFmt(lhs.span, "attempt to assign to const variable '{s}'", .{name});
                        return error.CodegenError;
                    }
                    // Discharge directly into the local's register — no MOVE.
                    try self.discharge2reg(val, reg);
                    return;
                }
                if (self.upvalues.get(name)) |idx| {
                    if (self.isConstUpvalue(idx)) {
                        self.setDiagFmt(lhs.span, "attempt to assign to const variable '{s}'", .{name});
                        return error.CodegenError;
                    }
                    const reg = try self.exp2anyreg(val);
                    _ = try self.builder.emitABC(.setupval, reg, idx, 0, line);
                    return;
                }
                if (self.outer != null) {
                    if (self.ensureUpvalue(name)) |idx| {
                        if (self.isConstUpvalue(idx)) {
                            self.setDiagFmt(lhs.span, "attempt to assign to const variable '{s}'", .{name});
                            return error.CodegenError;
                        }
                        const reg = try self.exp2anyreg(val);
                        _ = try self.builder.emitABC(.setupval, reg, idx, 0, line);
                        return;
                    } else |_| {}
                }
                try self.checkDeclaredGlobal(lhs.span, name);
                if (self.isConstGlobal(name)) {
                    self.setDiagFmt(lhs.span, "attempt to assign to const variable '{s}'", .{name});
                    return error.CodegenError;
                }
                const reg = try self.exp2anyreg(val);
                const kid = try self.builder.internString(name);
                try self.emitGlobalSet(kid, reg, line);
            },
            .Field, .Index => {
                // For field/index targets, discharge value to a register first.
                const val_reg = try self.exp2anyreg(val);
                try self.genSet(lhs, val_reg, line);
            },
            else => {
                self.setDiag(lhs.span, "invalid assignment target");
                return error.CodegenError;
            },
        }
    }
```
